Install MongoDB on the computer

'npm install mongodb' - Install's the driver for node to run with 

Run the MongoDP applicaiton by opening this folder /node/task-manager/ and running this command :
mongod --dbpath=/home/kevin/mongodb-data

This will run the mongoDB Server on and IP address and port

You can then open the MongoDB GUI from file:///home/kevin/robo3t-1.3.1-linux-x86_64-7419c406/bin/robo3t

MongoDB.js is a file that writes/reads to the database once it is up and running
